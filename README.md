## 프로젝트 소개 📓
* 가족 및 친구들과 매년 여행을 다니며 여행 계획을 공유하고 사진을 서로 다운로드 받는 번거로움을 줄이기 위해 제작하게 되었습니다.
* 컴퓨터 프로젝트 과제 - 소프트웨어 제작
